2Archie Hotel Concierge Bot

S Nicholson X00133895 IT Tallaght

- A college project test bot that uses the Cognitive Services LUIS API to determine which simple services to deliver over chat channels.

The bot is hosted in Azure. GitHub is used for both version control and continuous deployment. Development in Visual Studio 2017 IDE is pushed to a GitHub repository that alerts Azure to pull the new version as it becomes available.

The bot itself operates through the Bot Framework Bot Connector to handle interaction with messaging channels. It takes a natural language query and leverages the LUIS Cognitive Services.

A trained LUIS API interprets the user post and returns an Intent and Entity to the bot (as properties of serializable objects in the form of raw JSON data), which is used to determine response: for instance, the intent that the user wishes to perform a search using the Entity keyword.

Search queries and CRUD are conducted upon a remote SQL database, via LINQ, also hosted in Azure and operating independently as a service back-end. The query data returned to the bot is then presented in a rich card carousel format.
